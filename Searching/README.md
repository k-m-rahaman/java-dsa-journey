# 📖 About

Searching algorithms are used to locate an element inside a collection efficiently.

This module introduces both linear and advanced searching techniques commonly used in interviews and competitive programming.

---

# 📚 Programs Included

| No | Program | Time Complexity |
|----|----------|-----------------|
| 01 | Linear Search | O(n) |
| 02 | Binary Search | O(log n) |
| 03 | First Occurrence | O(log n) |
| 04 | Last Occurrence | O(log n) |
| 05 | Count Occurrences | O(log n) |
| 06 | Find Missing Number | O(n) |
| 07 | Peak Element | O(log n) |
| 08 | Jump Search | O(√n) |
| 09 | Exponential Search | O(log n) |
| 10 | Interpolation Search | O(log log n) (average) |
